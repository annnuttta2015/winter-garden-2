Зимний сад 2: Урожай Вупсильды

Структура архива для GitHub Pages:
- index.html
- assets/
- README.txt
- asset_sources.json

Что поправлено в этой сборке:
1. Верхняя статистика переписана на CSS-карточки, без зависимости от встроенных подписей panel_stats_main.png.
   Слева показываются крестики, справа на основном экране — банки, на экране лукошка — плоды.
2. На экране лукошка справа выводится “плодов”, например 11/100 плодов.
3. Лукошко собрано слоями: задняя корзина, слой плодов внутри корзины, передний край корзины поверх плодов.
4. Шкала прогресса использует progress_bar_bg.png и progress_bar_fill.png.
5. Кнопка “Скачать картинку” подключена через html2canvas. На GitHub Pages работает стабильнее, чем при открытии index.html локальным файлом.
6. Механика Вупсильды: шанс 10% съесть 100 крестиков, но не в первые 2 добавления.
7. Механика Вупсента: максимум 2 раза за игру, разбивает только банку с 1–4 плодами, сбрасывая её до 0.
8. Исправлен учёт плодовых “дропов”: если Вупсент разбил банку, потерянные плоды не восстанавливаются автоматически от уже добавленных крестиков.

Как залить на GitHub Pages:
1. Распаковать архив.
2. Загрузить index.html, папку assets, README.txt и asset_sources.json в корень репозитория.
3. Включить GitHub Pages для ветки/папки, где лежит index.html.

Если игра уже открывалась раньше, браузер может держать старый локальный прогресс. Для чистой проверки можно нажать “Управление прогрессом” → “Сбросить прогресс”.
